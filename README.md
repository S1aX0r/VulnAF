# VULNAF
VulnAF is a test website dedicated to AO3: Injections from the OWASP Top 10.

It must be noted that the directory that runs the Docker container must look like this:
```text
vuln-injections/
├── app.py
├── Dockerfile
├── requirements.txt
├── ldap_mock.py
├── data/
│   ├── users.xml
│   └── fake_mongo.py
├── templates/
│   └── index.html
```
